# Employee Management System

Full Stack Employee Management System built using React.js, Node.js, Express.js and MongoDB.

## Features

- JWT Authentication
- Role Based Login
- Employee Management
- Add Employee
- Edit Employee
- Delete Employee
- Admin Dashboard

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT
